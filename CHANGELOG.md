# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-22

### Added
- Initial release of PhotoViewer.
- Automatic USB storage scanning for images.
- Slideshow functionality with configurable intervals (10s, 15s, 20s).
- Random transition effects (Crossfade, Slide, Zoom & Fade).
- Remote control (D-pad) support for navigation and settings.
- EXIF metadata display (Shooting Date).
- Image rotation and saving (overwrite) capabilities.
- Physical file deletion from external storage.
- Real-time clock and date overlay.
- Support for Android 8.0 (API 26) and above.
- Immersive full-screen mode for Android TV.
